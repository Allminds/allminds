Creating Mind Maps

You can create new mind map by clicking “Create MindMap” button on the home page.

It will take you to the newly created mind map and now you         can start editing!!

Editing mind maps

Select node

Selected node has a gray background.

A node can be selected by navigation using arrow keys or by single mouse click.

Edit text

Double click on the node which you want to edit, a text box will pop up. When you are done editing, hit enter to save changes.

Edit long text

Double click on the node you want to edit. A prompt will appear for editing.

Add node

Sibling

To add new sibling node to a particular node, select that node and hit enter to add sibling.

Child

To add new child node to a particular node, select that node and hit tab to add sibling.

Delete Node

Select the node you want to delete and press Fn+delete

Cut subtree

Select node you want to cut and press Cmd+X

Copy subtree

Select node you want to cut and press Cmd+C

Paste subtree

Select node where you want to paste the subtree and press Cmd+V

Collaboration

To collaborate on a mindmap, share url of the mind map with people.

The changes of collaborators are visible to everyone.

Expand/Collapse

Hit space on selected node to collapse/expand its subtree.
